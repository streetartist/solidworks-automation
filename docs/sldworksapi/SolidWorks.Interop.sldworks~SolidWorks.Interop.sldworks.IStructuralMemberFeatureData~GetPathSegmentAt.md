Remarks

The body generated by this feature is not available when the feature is rolled back. Therefore, it is not necessary to call [IStructuralMemberFeatureData::AccessSelections](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IStructuralMemberFeatureData~AccessSelections.md) or [IStructuralMemberFeatureData::ReleaseSelectionAccess](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IStructuralMemberFeatureData~ReleaseSelectionAccess.md) before or after calling this method.
