#### Return Value

- in-process, unmanaged C++: Pointer to an array containing the hit entities (see **Remarks**)

- VBA, VB.NET, C#, and C++/CLI: Not supported

See [In-process Methods](sldworksapiprogguide.chm::/OVERVIEW/In-process_Methods.htm) for details about this type of method.

Remarks

This method only returns information if the swRayPtsOptsTOPOLS option was specified in the Options argument of [IModelDoc2::RayIntersections](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IModelDoc2~RayIntersections.md).

The array returned contains information about each entity hit of each ray on the given bodies. The topology hit can be coordinated with the hit point using the BodyIndex return value from [IModelDoc2::IGetRayIntersectionsPoints](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IModelDoc2~IGetRayIntersectionsPoints.md). You can determine the type of item hit (face, edge or vertex) by checking the IntersectionType return value in the IModelDoc2::IGetRayIntersectionsPoints.

Requirements

**Target Platforms:** Windows 7, Windows Vista SP1 or later, Windows XP SP3, Windows Server 2008 (Server Core not supported), Windows Server 2008 R2 (Server Core supported with SP1 or later), Windows Server 2003 SP2
