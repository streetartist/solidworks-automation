# ISecondaryMemberUpToMembersFeatureData Interface Methods

Help ID: `SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ISecondaryMemberUpToMembersFeatureData_methods`

Allows access to a secondary structure system up-to member that is created between a selected point and one or more primary structure system members or between one or more point-member pairs.
For a list of all members of this type, see [ISecondaryMemberUpToMembersFeatureData members](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ISecondaryMemberUpToMembersFeatureData_members.md).
