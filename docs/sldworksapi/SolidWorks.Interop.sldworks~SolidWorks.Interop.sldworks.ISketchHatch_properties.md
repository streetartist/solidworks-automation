# ISketchHatch Interface Properties

Help ID: `SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ISketchHatch_properties`

Represents an area hatch, which is inserted into a SOLIDWORKS drawing polygon or component face when you click Insert &gt; Annotations &gt; Area Hatch/Fill in a SOLIDWORKS drawing.
For a list of all members of this type, see [ISketchHatch members](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ISketchHatch_members.md).
