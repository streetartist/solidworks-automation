# ICommandTabBox Interface Methods

Help ID: `SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ICommandTabBox_methods`

Allows add-in applications to create CommandManager tab boxes and add them to a CommandManager tab. The add-in application must create and clean-up its own tab boxes.
For a list of all members of this type, see [ICommandTabBox members](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ICommandTabBox_members.md).
