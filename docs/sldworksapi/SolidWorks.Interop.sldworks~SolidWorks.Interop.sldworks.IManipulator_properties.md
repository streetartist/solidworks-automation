# IManipulator Interface Properties

Help ID: `SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IManipulator_properties`

Allows access to a manipulator, which is similar to the triad or the drag arrow (also called a handle), in a SOLIDWORKS part or assembly document.
For a list of all members of this type, see [IManipulator members](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IManipulator_members.md).

Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | [Name](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IManipulator~Name.md) | Gets and sets the name of a manipulator. |
| Property | [Options](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IManipulator~Options.md) | Gets or sets whether the manipulator is deleted when a component in an assembly is modified or deleted. |
| Property | [Selectable](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IManipulator~Selectable.md) | Gets or sets whether the manipulator can be selected in a PropertyManager page's selection box. |
| Property | [Visible](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IManipulator~Visible.md) | Gets or sets whether the manipulator is visible in this SOLIDWORKS part or assembly document. |
