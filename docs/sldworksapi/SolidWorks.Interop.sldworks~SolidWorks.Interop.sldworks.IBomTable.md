**IMPORTANT:** You can no longer insert IBomTable objects; you can now only insert [IBomTableAnnotation](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IBomTableAnnotation.md) objects. IBomTable objects are not and cannot be converted to IBomTableAnnotation objects. Use the IBomTable APIs for legacy BOM tables only.
