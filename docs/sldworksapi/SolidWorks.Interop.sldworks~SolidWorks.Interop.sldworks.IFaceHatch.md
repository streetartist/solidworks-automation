# IFaceHatch Interface

Help ID: `SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IFaceHatch`

Represents a cross-hatch, which is automatically added by SOLIDWORKS when you create a section view, aligned section view, break view, or detail view.
Represents a cross-hatch, which is automatically added by SOLIDWORKS when you create a [section view](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IDrSection.md), aligned section view, break view, or [detail view](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IDetailCircle.md).
