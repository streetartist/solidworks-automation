#### Property Value

Last error that occured in this design table as defined in [swDesignTableErrors\_e](ms-help://SolidWorks.Interop.swconst/SolidWorks/SOLIDWORKS.Interop.swconst~SOLIDWORKS.Interop.swconst.swDesignTableErrors_e.html)

Remarks

Getting this property returns the last error generated by a request to modify the design table. Setting this property allows you to clear the last design table error that occurred.
