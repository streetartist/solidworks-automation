#### Return Value

Array containing the hit points (see **Remarks**)

Remarks

The array returned contains information about each hit of each ray on the given bodies, in the following format:

For each intersection found:

- (double) BodyIndex: index in BodiesIn array of the body that was hit
- (double) RayIndex: index in VectorsIn array of rays
- (double) IntersectionType: type of intersection found. See [swRayPtsResults\_e](ms-help://SolidWorks.Interop.swconst/SolidWorks/SOLIDWORKS.Interop.swconst~SOLIDWORKS.Interop.swconst.swRayPtsResults_e.html) . Entry/Exit information (swRayPtsResultsENTER and swRayPtsResultsEXIT) are only returned when swRayPtsOptsENTRY\_EXIT is specified in the options argument in the [IModelDoc2::RayIntersections](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IModelDoc2~RayIntersections.md) call. If more than one intersection type is found, then the IntersectionType contains an OR'd set of values from the swRayPtsResults\_e enumeration.
- (double) x,y,z: coordinates of hit point
- (double) nx,ny,nz: normal (optionally appears when swRayPtsOptsNORMALS is specified in the Options argument in the IModelDoc2::RayIntersections call).

Requirements

**Target Platforms:** Windows 7, Windows Vista SP1 or later, Windows XP SP3, Windows Server 2008 (Server Core not supported), Windows Server 2008 R2 (Server Core supported with SP1 or later), Windows Server 2003 SP2
