###### New functionality

#### Other major enhancements

- Import annotations into drawings. See [IView::ImportAnnotations](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IView~ImportAnnotations.md).
- New IRenderMaterial properties. See [New delegates, methods, and properties](#NewMethods0).
- Add or edit floor appearances of model scenes. See [ISwScene::AddOrEditFloorAppearance](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ISwScene~AddOrEditFloorAppearance.md) and [ISwScene::GetFloorAppearance](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ISwScene~GetFloorAppearance.md).
- Support for texture mapping of non-linear types, including surface projections. See [IFace2::GetTessTriStripTextures](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IFace2~GetTessTriStripTextures.md) and [IPartDoc::GetTessTriStripTextures](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IPartDoc~GetTessTriStripTextures.md).
- Improved performance when reloading a SOLIDWORKS model from disk by reloading only selected references. See [IModelDocExtension::ReloadWithReferences](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IModelDocExtension~ReloadWithReferences.md).
